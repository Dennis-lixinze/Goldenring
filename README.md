# Goldenring
A Qt based 3D RPG game
### 游戏介绍
1. 基本功能
   - 登陆界面：   登陆界面提供注册连接
   - 注册登陆：   提供用户密码登陆的功能
   - 角色的基本运动控制
      1. 在3D视图下实现玩家的前后左右运动
      2. 实现碰撞检测
      3. 实现上下楼梯等运动
      4. 实现简单的对战场景
2. 拓展功能
   - 添加网络模块：  
        单核服务器上可实现少于六人的同时在线，玩家之间互相可见
   - 用OPenGL搭建3D场景：  
        使用Qt封装的opengl库和glew库实现模型加载，游戏界面在Glwidget中渲染
